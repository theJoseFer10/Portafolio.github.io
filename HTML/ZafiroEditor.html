Zafiro Editor

Un editor de codigo en terminal creado en python con la libreria curses, el cual cuenta con funciones basicas como abrir, guardar, editar y cerrar archivos de texto plano. Ademas de contar con funciones adicionales como el resaltado de sintaxis para varios lenguajes de programacion.

Tecnologías utilizadas.

  • Python.
  • Curses
